# 🏨 Book My Stay

A modern and intuitive hotel booking web application that enables users to explore accommodations, view detailed listings, and make reservations seamlessly.

🌐 **Live Demo:** https://book-my-stay-ruddy.vercel.app

---

## ✨ Features

- 🔍 Browse hotels and available stays
- 🏡 Detailed property pages with essential information
- 📅 Smooth and user-friendly booking experience
- 📱 Fully responsive design (mobile, tablet, desktop)
- ⚡ Fast performance with optimized UI

---

## 🛠️ Tech Stack

- **Frontend:** React / Next.js  
- **Styling:** Tailwind CSS / CSS  
- **Deployment:** Vercel
